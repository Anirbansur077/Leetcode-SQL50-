# 1757. Recyclable and Low Fat Products

## Problem

Find the IDs of products that are both recyclable and low fat.

## SQL Concepts Used

- WHERE
- AND operator
- Filtering rows

## Approach

Filter the table using two conditions:
- low_fats = 'Y'
- recyclable = 'Y'
